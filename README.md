# Unity Scene MJCF Exporter

This is a Unity Editor tool for exporting Unity scenes or selected GameObjects to MuJoCo MJCF XML.

## Usage

After importing the package, use the Unity menu:

- Gewu > MuJoCo > Export Current Unity Scene To MJCF XML
- Gewu > MuJoCo > Export Selected GameObjects To MJCF XML

## Requirements

This tool depends on Unity Robotics URDF Importer if your scene contains URDF robot components.
